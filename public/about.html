The Project

At a research-focused school like UC Davis, programming knowledge can be extremely helpful to students and researchers. It can be used for data visualization, data analysis, or even just to automate repetetive tasks. However, programming has long been an esoteric field. It seems like there are people with the knowledge, and those without. I firmly believe that there is no more reason for such a gap. I thought of this project as a way to help people learn to code, even if it's been previously inaccessible.
